# 🚀 Databricks SQL AI Functions: The "AI 12" Playground

A comprehensive collection of the 12 essential **Databricks AI Functions**. This repository demonstrates how to integrate Large Language Models (LLMs) directly into SQL workflows to perform advanced data tasks without leaving the data warehouse.

---

## 📖 Overview

This project showcases how Databricks has unified AI and Data Warehousing. By using these functions, data engineers and analysts can apply generative AI to tables at scale—handling everything from sentiment analysis and language translation to structured data extraction and PII masking.

### 🛠️ Prerequisites
* **Databricks SQL Warehouse**: Pro or Serverless tier.
* **Permissions**: Access to Foundation Model APIs (e.g., Llama 3, Mixtral).

---

## 🧪 The "AI 12" Covered

| Function | Use Case |
| :--- | :--- |
| **`ai_query`** | The "Jack of all trades" for custom LLM prompts. |
| **`ai_summarize`** | Condenses long-form text into concise summaries. |
| **`ai_translate`** | Seamlessly translates text across multiple languages. |
| **`ai_analyze_sentiment`** | Automatically detects the "vibe" (positive/negative/neutral). |
| **`ai_classify`** | Categorizes data into custom labels (Zero-shot). |
| **`ai_extract`** | Pulls structured entities (emails, names) out of messy text. |
| **`ai_fix_grammar`** | Corrects syntax and improves readability. |
| **`ai_gen`** | Quick-fire text generation for metadata. |
| **`ai_mask`** | Redacts sensitive PII (Names, CC numbers, etc.). |
| **`ai_similarity`** | Measures semantic "closeness" between strings. |
| **`ai_forecast`** | Predicts future trends in time-series data. |
| **`ai_parse_document`** | Extracts insights from unstructured PDFs and images. |
